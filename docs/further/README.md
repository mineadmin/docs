# 后台

## 布局
MineAdmin提供了三种布局方式，都支持自适应，可选择合适自己的布局模式。

<img src="https://s1.ax1x.com/2022/08/19/vr0fAJ.png" />

:::tip
其实可以自行开发扩展出新的布局
:::

## 换肤
我们提供了三种皮肤样式，每种皮肤都对布局做了适配，可自行选择自己喜欢的皮肤。

- 皮肤与整体换色是不同的，未来也会增加整体换色功能

<img src="https://s1.ax1x.com/2022/08/19/vr0LHe.png" />

:::tip
皮肤也是可以扩展的
:::
